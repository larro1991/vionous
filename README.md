### Use a Knowledge Package

```bash
# Clone the repo (uses Git LFS for training data)
git clone https://github.com/larro1991/vionous.git

# Navigate to any package
cd vionous/packages/math/training-data/

# Training data ready to use
ls -la
# train.jsonl (1.1M pairs)
# val.jsonl (123K pairs)
```

### Train a LoRA Adapter

```python
# Example using PEFT
from peft import LoraConfig, get_peft_model
from transformers import AutoModelForCausalLM

# Load base model
model = AutoModelForCausalLM.from_pretrained("Qwen/Qwen2.5-7B-Instruct")

# Configure LoRA
lora_config = LoraConfig(
    r=16,
    lora_alpha=32,
    target_modules=["q_proj", "k_proj", "v_proj", "o_proj"],
    lora_dropout=0.05,
)

# Train with your favorite framework (Axolotl, Unsloth, etc.)
```
